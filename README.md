# TaskTimer

A beautiful, minimalist task timer application built with React.

## Features

- Create and manage daily tasks
- Set timers for each task
- Track completion history
- Favorites system for recurring tasks
- Backlog for future tasks
- Analytics dashboard
- Dark mode support
- Task history

## Getting Started

This project is ready to deploy to Vercel. Follow these steps:

### Prerequisites

- Node.js 18+ installed on your computer
- A GitHub account
- A Vercel account

### Local Development

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:5173`

### Deploy to Vercel

1. Push this code to your GitHub repository
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select your GitHub repository
5. Click "Deploy"

That's it! Your app will be live in under a minute.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
